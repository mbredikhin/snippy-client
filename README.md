# Snippy-client

Front-end for [snippy](https://github.com/mbredikhin/snippy.git) app

## Run Locally

Clone the project

```bash
  git clone https://github.com/mbredikhin/snippy-client.git
```

Go to the project directory

```bash
  cd snippy-client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
